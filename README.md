# Simple Order

## サイト概要
発注を一元管理し、業者とのマッチングもできます。

### サイトテーマ
発注をより楽に、分かりやすく、効率的に！

### テーマを選んだ理由
ネット環境が当たり前になった時代においても、いまだにFAX発注や電話発注の業者は多く、その度に手間がかかるので、一つのアプリケーション上で全て完結できたら業務の効率化に繋がると思ったからです。

### ターゲットユーザ
飲食店オーナーや従業員、食材卸業の方々（業者）

### 主な利用シーン
飲食店従業員が食材や備品の発注をする際に利用。
